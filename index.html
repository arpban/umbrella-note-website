Our goal with Umbrella Note has been to build a beautiful and extremely simple application for taking notes and keeping journals.

Minimal user interface

Works Offline

Syncs data with Dropbox, Google drive or Onedrive.

Work in progress 🚧 Available on multiple platforms

Keyboard Shortcuts

Work in progress 🚧 Integration with Google assistant

Work in progress 🚧 Smart text editor - spell check, dictionary, word suggestions

User keeps all the data. None of the data is stored on Umbrella Note's servers.

User can carry their data in a USB drive

Application will work with all its features even if Umbrella Note shuts down

Themes, Night Mode
